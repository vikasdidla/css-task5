combinater selector:

1.  Decendent Selector

The descendant combinator matches all elements that are descendants of a specified element.

Example

  div   p {
              background-color: yellow;
            }

2.  child Selector

The child combinator selects all elements that are the children of a specified element.

Example

  div p > {
              background-color: yellow;
            }

3.  Ajacent sibling Selector

The next sibling combinator is used to select an element that is directly after another specific element. Sibling elements must have the same parent element, and "adjacent" means "immediately following".

Example

  div p+{
              background-color: yellow;
            }

4. General sibling Selector

The General sibling/subsequent-sibling combinator selects all elements that are next siblings of a specified element.

Example

  div p~{
              background-color: yellow;
            }
